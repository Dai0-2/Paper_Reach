# Roadmap

## Near-Term

- add Semantic Scholar backend
- add Crossref backend
- improve metadata deduplication
- support DOI resolution and DOI-first local inputs
- expand sample fixtures and demo outputs

## Full-Text Improvements

- richer section extraction from PDFs
- GROBID integration
- better table and figure context capture
- stronger evidence localization by section heading

## Ranking And Analysis

- configurable rubric weights
- richer exclusion-rule handling
- explainable gap-analysis templates
- better query expansion suggestions from rejected and ambiguous sets

## Evaluation

- benchmark dataset for paper-screening tasks
- regression suite for ranking consistency
- offline replay fixtures for scholarly API responses

## Ecosystem

- tighter integration patterns for Codex, Claude Code, Cursor, and similar coding agents
- export adapters for review tools and note-taking pipelines
- shared skill registry examples for literature workflow automation
